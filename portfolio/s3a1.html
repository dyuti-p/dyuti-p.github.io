Assignment Guidelines

OVERVIEW
Individually or in small groups of 2 or 3, students will design and write a program or script to be performed. This can be taken literally, as in your script is intended to be enlivened by actors. This can be taken loosely and all terms can be creatively interpreted. Regardless, all performances must incorporate learned principles of scripted “functions,” and must also include an element of “chance.”

GOALS
Plan — All scripts must include a written component. Will this be a literal script for actors? A set of instructions to be performed? A set of rules that can inform a dynamic experience? (5 pts)
Script — Your script must creatively interpret the logic of learned markup and programming languages, but does not need to be syntactically correct. (10 pts)
Execute — All performances must be able to be enacted for a minimum of 2 minutes (typically: 1 minute per page in a standard screenplay). If your duration exceeds 5 minutes, the entirety or continuation of your script must be adequately described or summarized. How will you critically and creatively engage your audience? (10 pts)
Document — You will create a website that appropriately documents your performance. (5 pts)
