Assignment Guidelines

OVERVIEW
Using p5.js, students will program a generative design that employs functions, loops, and conditionals. Designs must also incorporate an element of randomness or chance in their code. Students will work to embed this design within a separate website that also incorporates designed and user-interactive elements.

GOALS
Get messy and have fun while exploring the creative potential within generative design tactics.
Incorporate conditional loops (for, if/else, while) with randomly defined parameters or arguments to produce a unique design each time a user loads the page. (15 points)
Seamlessly integrate at least one object or element that is changed or manipulated by user feedback (15 points).
Creatively combine your p5.js script with HTML and CSS to style, display, and permit further interactivity with pseudo-class selectors (:hover, :active, :focus, :target, :not(), :has(), etc). (10 points)
Be able to comfortably explain your concept, process, and code (line by line) in a critique setting. (10 pts)
