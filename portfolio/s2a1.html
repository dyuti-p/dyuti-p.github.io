Assignment Guidelines

OVERVIEW
As a class, students will work together to design and build a Web Ring that links and loops between all students’ web publications (Section 1 Final Project). The Web Ring may be an external site of links (an index), but must also incorporate some form of modular code that is implemented on each student’s publication site.

GOALS
Work together to build and design a modular code that links between all students' publications. Your design must accommodate all pre-existing web publication designs in a seamless and intuitive way, retroactively reassigning/discovering hierarchy of information (10 points)
Revisit your individual web publications to deploy the modular code as a feature update/patch (5 points)
