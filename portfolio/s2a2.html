Assignment Guidelines

OVERVIEW
Students will create a self-contained, single-page site that employs hover effects and CSS selectors to assist and encourage user interaction. Your site must include a minimum of 5 images (observing optimization and best practices) and all elements must contain adequate metadata/alt text. Your site must also account for viewport sizes to serve comparable or alternate user experiences.

GOALS
Creatively utilize extended HTML and CSS options to account for users’ individual environments while mindfully serving appropriate content and styling. The full load of your page must be no larger than 1mb (5 pts).
All sites must demonstrate:
CSS Selectors to control separate elements or classes based on user feedback (:hover, :active, :focus, etc) (10 pts)
Viewport size accommodations by either solely using viewport ratio properties (for positioning, text size, etc) or serving entirely separate content to different users with @media rules (minimum of 3) (10 pts)
Alt text/title/metadata to offer fallback content options or alternative experiences (minimum of 5 images) (5 pts)
